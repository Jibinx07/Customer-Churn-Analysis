# 📊 Customer Churn Analysis & Prediction

## 📌 Project Overview
This project analyzes customer churn behavior in a telecom company and builds a machine learning model to predict customer churn.

---

## 🎯 Objective
- Identify factors influencing customer churn  
- Analyze customer behavior  
- Build a predictive model  

---

## 🛠️ Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Power BI
- Machine Learning (Logistic Regression)

---

## 📊 Dashboard


---

## 💡 Key Insights
- Month-to-month contracts have the highest churn rate  
- Higher monthly charges increase churn risk  
- Long-term customers are more loyal  
- Payment methods influence churn behavior  

---

## 🤖 Machine Learning
- Model: Logistic Regression  
- Accuracy: ~75%
